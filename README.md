# Machine Learning Project By Data Genes
The goal of this project is to build a Market Basket Analysis & Product Recommendation System using Instacart e-commerce data.

---

## Steps to clone the project :
1. Type : git clone https://github.com/samitochi04/machine_learning_project in a terminal to clone the repository.
2. Create a folder called **data** and download the dataset from **https://www.kaggle.com/datasets/yasserh/instacart-online-grocery-basket-analysis-dataset/data**, extract the zip folder in the data folder.
3. Explore the project_structure.md to understand the project structure and code to get started.
